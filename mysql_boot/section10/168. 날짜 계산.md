## 날짜 계산

- DATEDIFF()
  - 두 날짜를 가져와 차이를 계산한다
  - 일 수로 알려준다
  - DATEDIFF('2007-12-31 23:59:59', '2007-12-30');
- DATE_ADD()
  - 지정한 날짜에 특정 Format으로 지정된 시간을 더한다.
  - DATE_ADD('2007-12-31 23:59:59', INTERVAL 1 DAY);
  - DATE_ADD('2007-12-31 23:59:59', INTERVAL '1:1' MINUTE_SECOND);
  - '+'와 '-' 도 쓸 수 있다

`Mysql에서 DATE로 시작하는 함수는 datetime도 대부분 받는 함수 이다`

```sql
SELECT name, birthdate, DATEDIFF(NOW(), birthdate) AS diff_day FROM people;
+---------+------------+----------+
| name    | birthdate  | diff_day |
+---------+------------+----------+
| Padma   | 1983-11-11 |    14202 |
| Larry   | 1943-12-25 |    28768 |
| Toaster | 2022-09-29 |        0 |
+---------+------------+----------+

SELECT birthdt, DATE_ADD(birthdt, INTERVAL 1 MONTH) FROM people;
+---------------------+-------------------------------------+
| birthdt             | DATE_ADD(birthdt, INTERVAL 1 MONTH) |
+---------------------+-------------------------------------+
| 1983-11-11 10:07:35 | 1983-12-11 10:07:35                 |
| 1943-12-25 04:10:42 | 1944-01-25 04:10:42                 |
| 2022-09-29 15:25:04 | 2022-10-29 15:25:04                 |
+---------------------+-------------------------------------+

SELECT birthdt, DATE_ADD(birthdt, INTERVAL 10 SECOND) FROM people;
+---------------------+---------------------------------------+
| birthdt             | DATE_ADD(birthdt, INTERVAL 10 SECOND) |
+---------------------+---------------------------------------+
| 1983-11-11 10:07:35 | 1983-11-11 10:07:45                   |
| 1943-12-25 04:10:42 | 1943-12-25 04:10:52                   |
| 2022-09-29 15:25:04 | 2022-09-29 15:25:14                   |
+---------------------+---------------------------------------+

SELECT birthdt, birthdt + INTERVAL 1 MONTH FROM people;
+---------------------+----------------------------+
| birthdt             | birthdt + INTERVAL 1 MONTH |
+---------------------+----------------------------+
| 1983-11-11 10:07:35 | 1983-12-11 10:07:35        |
| 1943-12-25 04:10:42 | 1944-01-25 04:10:42        |
| 2022-09-29 15:25:04 | 2022-10-29 15:25:04        |
+---------------------+----------------------------+

SELECT birthdt, birthdt - INTERVAL 5 MONTH FROM people;
+---------------------+----------------------------+
| birthdt             | birthdt - INTERVAL 5 MONTH |
+---------------------+----------------------------+
| 1983-11-11 10:07:35 | 1983-06-11 10:07:35        |
| 1943-12-25 04:10:42 | 1943-07-25 04:10:42        |
| 2022-09-29 15:25:04 | 2022-04-29 15:25:04        |
+---------------------+----------------------------+

SELECT birthdt, birthdt - INTERVAL 5 MONTH + INTERVAL 3 MONTH FROM people;
+---------------------+-----------------------------------------------+
| birthdt             | birthdt - INTERVAL 5 MONTH + INTERVAL 3 MONTH |
+---------------------+-----------------------------------------------+
| 1983-11-11 10:07:35 | 1983-09-11 10:07:35                           |
| 1943-12-25 04:10:42 | 1943-10-25 04:10:42                           |
| 2022-09-29 15:25:04 | 2022-07-29 15:25:04                           |
+---------------------+-----------------------------------------------+
```
