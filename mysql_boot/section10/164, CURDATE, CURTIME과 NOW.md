## CURDATE, CURTIME과 NOW

- CURDATE
  - 현재 날짜
- CURTIME
  - 현재 시각
- NOW
  - 현재 날짜 시각

```sql
SELECT CURDATE();
+------------+
| CURDATE()  |
+------------+
| 2022-09-27 |
+------------+

SELECT CURTIME();
+-----------+
| CURTIME() |
+-----------+
| 15:37:25  |
+-----------+

SELECT NOW();
+---------------------+
| NOW()               |
+---------------------+
| 2022-09-27 15:37:41 |


INSERT INTO people(name, birthdate, birthtime, birthdt)
VALUES('Toaster', CURDATE(), CURTIME(), NOW());

SELECT * FROM people;
+---------+------------+-----------+---------------------+
| name    | birthdate  | birthtime | birthdt             |
+---------+------------+-----------+---------------------+
| Padma   | 1983-11-11 | 10:07:35  | 1983-11-11 10:07:35 |
| Larry   | 1943-12-25 | 04:10:42  | 1943-12-25 04:10:42 |
| Toaster | 2022-09-27 | 15:39:13  | 2022-09-27 15:39:13 |
+---------+------------+-----------+---------------------+
```
